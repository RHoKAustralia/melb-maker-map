The Maker Network

Join the Maker Movement

The Maker Network is an ongoing open source project which aims to create a global database of maker resources - from workspaces to incubators - which is powered by the maker community and easily searchable on any device. If you would like to help you can start by mapping a resource, joining the project discussion or checking out the map source on github.
